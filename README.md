## 🚀 Deployment

### GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to **Deploy from branch** → `main` → `/ (root)`
4. Your site goes live at `https://ayman-26-dev.github.io/cv-en-ligne-ayman/`
